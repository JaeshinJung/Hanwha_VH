# General Rules
- 모든 설명과 답변은 반드시 한국어로 작성할 것.
- Write all code comments in English.
- 작업을 시작하기 전에 반드시 [plan.md](file:///c:/Users/USER/Desktop/hanwha_VH/plan.md) 파일을 확인하고, 그곳에 적힌 개발 계획과 순서에 따라 코드를 작성할 것.
- 비개발자가 진행하는 프로젝트이므로, 한 번에 너무 많은 코드를 수정하지 말고 작고 확인 가능한 단위로 변경할 것.
- 화면 디자인은 모바일 환경에서도 잘 보이도록 반응형(Responsive)으로 깔끔하게 구성할 것.
- 에러가 발생했을 때는 코드부터 수정하지 말고, 에러의 원인을 한국어로 먼저 쉽게 설명한 뒤 해결책을 제시할 것.
